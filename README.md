# JChat - Anonymous Random Chat App

JChat (Chatrix) is now a **Single-File** anonymous chat application. No Node.js, no React, and no folders required!

## 🚀 How to set up

1. **Firebase Keys:** Open `index.html` in a text editor and replace the placeholder keys with your own Firebase keys.
2. **Firebase Setup:**
   - In Firebase Console, enable **Anonymous Auth**.
   - In Firebase Console, create a **Firestore Database** in Test Mode.
3. **Deployment:** Simply upload **`index.html`** and **`logo.png`** to your Hostinger `public_html` folder.

## ✨ Features
- 🛡️ 100% Anonymous.
- ⚡ Instant Random Matching.
- 🚀 Real-time Messaging.
- 🎨 Modern Premium Design.
